  - 키워드 알림 설정 및 경험치 추적 기능 설정

1. 키워드 알림 설정
  - 키워드 입력 시, 색상 및 알림음으로 강조 효과
  - @단어1, @단어2 입력
  - Ex) @드드 -> @드드가 포함된 채팅 발견 시, 알림
2. 경험치 추적
  - 채팅 로그의 경험치를 취합하여, 총 획득한 경험치 및 시간당 획득한 경험치를 추산하는 기능
  - 활성화 시, 새로운 창이 생성되고 X를 누르면 초기화 가능

1. 채팅 필터 및 색상
   - 일반, 팀, 클럽, 외치기, 시스템 중 체크한 채팅 로그만 표기
   - 색상 팔레트를 이용하여 출력되는 색상 변경
2. 텍스트 설정
   - 기본 폰트 2종, 사용자 폰트 1종 사용 가능
   - 사용자 폰트를 사용하기 위해선, 프로그램 실행파일이 있는 폴더에 Font 폴더를 만들어 UserDefine.ttf 파일의 폰트를 넣어주면 사용 가능
   - 폰트 크기는 슬라이드 바를 이용하여 조절 가능
3. 위치 설정
   - 기본적으로 테일즈위버 이름을 가진 창의 중앙 하단부에 위치하며, PC에 따라 사용자의 취향에 따라 위치를 변경할 수 있음
   - X좌표의 -값을 넣으면 왼쪽, +값을 넣으면 오른쪽으로 이동
   - Y좌표의 -값을 넣으면 왼쪽, +값을 넣으면 오른쪽으로 이동
   - 최대값은 테일즈위버 이름을 가진 창의 크기에 비례하여 움직일 수 있음
       창이 사라져 컨트롤을 못하는 경우를 방지하기 위해 최대값이 있음
